# Fract-ol
This is a project for Shcool42.
You will need a homebrew MLX library in order to run it.

Controls:

Q - decrease increments

E - increase increments

R - restart fractol

TAB - change color

1 - Mandelbrot

2 - Julia

3 - Burning Ship

4 - Cluster

Arrows - move

Mouse Wheel - Zoom In/Zoom out

To run this programm:

1. make
2. ./fractol mandelbrot

Enjoy:D
